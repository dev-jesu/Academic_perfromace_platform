# UI/UX Wireframes Prompt for Stitch

## Prompt
Create a set of high‑fidelity wireframes for a **College Academic Performance Assessment Platform**. The wireframes should cover the following screens and user journeys:

1. **Dashboard Overview** – Shows summary cards for overall CGPA, current semester GPA, attendance percentage, and upcoming assessments.
2. **Student Profile** – Detailed view of student information, list of enrolled courses, grades per course, and a visual grade distribution chart.
3. **Course Detail** – Course information, syllabus link, list of assignments/exams with scores, and a progress bar for the semester.
4. **Mentor‑Mentee Management** – Mentor profile, list of assigned mentees, communication panel, and mentorship timeline.
5. **Assessment Entry** – Form for entering a new assessment result (type, score, date) with validation rules.
6. **Admin Settings** – Manage users, define grading scales, configure academic periods, and view system health metrics.

### Design Requirements
- **Style**: Modern, clean, and premium look with a dark‑mode base, vibrant accent colors (e.g., teal and amber), glass‑morphism cards, and subtle micro‑animations on hover.
- **Typography**: Use the "Inter" font family from Google Fonts for headings and body text.
- **Color Palette**:
  - Background: `#121212`
  - Primary Accent: `#00bfa5`
  - Secondary Accent: `#ffb300`
  - Text: `#e0e0e0`
- **Components**: Consistent navigation sidebar, top bar with user avatar, responsive layout for tablets and desktops.
- **Interaction**: Include tooltips for icons, smooth transition when switching tabs, and loading skeletons for data tables.

### Output Format
Provide the prompt as a plain text block that can be copied into Stitch (or any AI‑driven UI generator). Include clear section headings and bullet points as shown above.
